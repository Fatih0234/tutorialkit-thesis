# Interactive Counter Lab

Build a small JavaScript counter while learning how state, rendering, and DOM events work together.

## Learner challenge

1. Complete the decrement handler without allowing negative values.
2. Complete the reset handler so the counter returns to zero.
3. Test both changes in the live preview.
4. Save your experiment before returning to the lecture.

Select either incomplete handler and use **Ask AI about selection** if you want a hint without leaving your workspace.
